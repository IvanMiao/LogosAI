欢迎使用 LogosAI！🚀

LogosAI 是一款智能语言处理工具，可以帮助您深入阅读和理解任何语言的文本。无论是图片、PDF还是纯文本，LogosAI 都能帮您提取文字、进行智能校对、深入解读并翻译成您需要的语言。

## 主要功能 ✨

1.  **文字识别 (OCR) 与校对**：
    *   从图片、PDF 文件或直接从文本文件中提取文字。
    *   使用 AI 技术对提取的文字进行校对，修正可能的错误和格式问题。
2.  **文本解读**：
    *   对文本进行深度分析，包括语法、修辞和文化背景。
    *   目前支持“通用”、“新闻”和“哲学思辨”等模式，更多模式敬请期待！
3.  **文本翻译**：
    *   将文本准确翻译成多种语言，包括德语、英语、法语、俄语和中文等。

## 如何使用 LogosAI 📖

打开 LogosAI 应用后，您会看到一个简洁的界面。以下是主要操作步骤：

### 1. 配置 API 密钥

在使用各项高级功能之前，您需要提供 API 密钥。

*   在页面顶部的 **"API Configuration"** (API 配置)区域：
    *   **Mistral API Key**: 输入您的 Mistral API 密钥。这个密钥主要用于文字识别 (OCR) 和文本处理。
    *   **Gemini API Key**: 输入您的 Gemini API 密钥。这个密钥主要用于文本的深度解读和翻译。
    *   输入完毕后，点击 **"Save keys"** (保存密钥) 按钮。您会看到一条确认信息。

### 2. 文本处理

完成 API 配置后，您可以开始处理文本了。点击 **"Text"** (文本) 标签页。
如果您不需要上传文件，您可以直接把想要解读的文本粘贴进文本框，然后跳过这一步骤，直接进行文本解读。

*   **上传文档**:
    *   在左侧，点击 **"Upload Image/PDF/text"** (上传图片/PDF/文本) 区域来选择您要处理的文件。支持图片格式、PDF 文件和 .txt 文本文件。
*   **处理文件**:
    *   上传文件后，点击 **"1. File Process (OCR/Read)"** (文件处理(OCR/读取)) 按钮。
    *   处理完成后，提取出的原始文本会显示在右侧 **"Raw Text(editable)"** (原始文本(可编辑)) 的文本框中。您可以在这里手动编辑文本。
    *   同时，初步格式化后的文本会显示在 **"Formatted Text"** (格式化文本) 标签页下。
*   **AI 校对**:
    *   如果您希望 AI 进一步修正原始文本中的潜在错误（比如 OCR 识别错误或格式问题），可以点击 **"2. AI Correct"** (AI 校对) 按钮。
    *   校对后的文本会更新在 **"Raw Text(editable)"** 和 **"Formatted Text"** 中。

### 3. 文本解读 🎓

当您有了处理好的文本后，可以进行深度解读。点击 **"🎓 Interpretation"** (解读) 标签页。

*   **配置解读设置**:
    *   **Prof's Language** (教学语言): 选择解读内容主要使用的语言（例如：EN 代表英语，ZH 代表中文）。
    *   **Language to Learn** (学习语言): 选择您希望学习和分析的文本的原始语言。
    *   **Genre** (文体类型): 选择文本的类型，例如 "General" (通用), "News" (新闻), "Philosophy" (哲学思辨)。不同的类型会采用不同的分析策略。
*   **生成解读**:
    *   设置完成后，点击 **"Generate Interpretation"** (生成解读) 按钮。
    *   解读结果会显示在右侧的 **"COURSE"** (课程)区域的 Markdown 框内。

### 4. 文本翻译

如果您需要将文本翻译成其他语言，请点击 **"Translation"** (翻译) 标签页。

*   **配置翻译设置**:
    *   **Target Language** (目标语言): 从下拉列表中选择您希望将文本翻译成的语言，例如 "中文", "English", "Français" 等。
*   **开始翻译**:
    *   选择目标语言后，点击 **"Translate!"** (翻译!) 按钮。
    *   翻译结果会显示在右侧的 **"Translation Result"** (翻译结果) 区域的 Markdown 框内。
